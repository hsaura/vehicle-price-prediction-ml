# Vehicle Price Prediction using Machine Learning

## Overview
This project explores machine learning techniques for predicting vehicle prices using structured automotive data.

The notebook covers:
- Data preprocessing
- Feature engineering
- Regression modeling
- Model evaluation
- Hyperparameter experimentation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- Matplotlib

## Motivation
This project was developed as part of my Master's studies in Software Engineering (AI) and helped deepen my understanding of practical machine learning workflows and predictive modeling.

## What I Learned
Through this project I improved my understanding of:
- handling structured datasets
- normalization techniques
- feature selection
- regression model optimization
- evaluating model performance
